How to install 
Step 1 | Download the file
Step 2 | Open the OpenME.rar File
Step 3 | Right click on your fivem
Step 4 | Open File location
Step 5 | Open the file called FiveM Application Data
Step 6 | Find the file called Plugins if you dont see it make one. This is what mine looks like C:\Users\bap11\AppData\Local\FiveM\FiveM.app\plugins
Step 7 | Drag the SirenSetting_Limit_Adjuster.asi file into your plugins 
Step 8 | Close all of that 
Step 9 | Close your FiveM if you have it open 
Step 10 | Your Done open your Fivem and test it
